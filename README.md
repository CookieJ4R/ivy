ivy - inventory manager

ivy is a inventory manager that is inspired by homebox and similar applications but ~worse~ with way less functionality by design. Im mainly developing it for my own use because I want something simpler than existing apps that fits my usecase better, but hope to release it in the future for other "likeminded" people.

Currently ivy is in an early mvp/prototyping stage where basic features are implemented but not yet that refined:
- Item creation (no deletion or editing tho)
- Tag crud
- Location crud

Planned:
- ci/cd build and release
- Item edit/delete features
- search functionality
- dashboard
- multi-stage locations (e.g. room -> shelf -> drawer)
- inventory export
- lending items
- maybe qr code printing and scnaning (low prio)
- better ui/ux
